# Example Design Note

This is a minimal example of a Markdown source note that is linked to a long-lived Google Doc for review.

## Problem

We need a workflow where comments happen in Google Docs but source edits happen in Markdown.

## Approach

- Publish Markdown into a linked Google Doc.
- Pull comment threads back into the source workflow.
- Apply accepted feedback to the Markdown file.
- Reply to and resolve the corresponding review threads.
